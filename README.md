# Music Player (School Activity)

A simple web-based music player built as a school activity using **HTML**, **CSS**, and **JavaScript**. This project allows users to play favorite songs, view album covers, browse synchronized lyrics from external lyric files, and interact with a responsive music player interface across different screen sizes.

---

## Overview

This project was created to practice front-end development fundamentals while building a functional and visually engaging music player. It focuses on organizing assets properly, handling audio playback, managing dynamic content with JavaScript, and creating a responsive layout that works on desktop and mobile devices.

The application includes song navigation, album artwork display, lyric rendering, lyric highlighting synchronized with the current song, an animated visualizer effect while music is playing, and a footer with proper attribution and copyright notice.

---

## Features

- Play and pause music
- Previous and next song controls
- Dynamic song title and album cover updates
- Lyrics loaded from external `.txt` files
- Synchronized lyric highlighting based on playback time
- Animated player beats while audio is playing
- Responsive layout for desktop, tablet, and mobile
- Footer with attribution and copyright notice
- Organized folder structure for easier maintenance

---

## Technologies Used

- **HTML5** for page structure
- **CSS3** for styling, responsiveness, layout, and animations
- **JavaScript** for audio controls, DOM manipulation, lyric loading, and lyric synchronization
